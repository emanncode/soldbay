### `soldbay-app` — NativeWind + Expo design system setup

**Installed packages:**
- `nativewind@4.2.6`, `tailwindcss@3.4.19`, `react-native-reanimated`, `react-native-safe-area-context`, `react-native-css-interop`
- `@expo-google-fonts/bricolage-grotesque@0.4.1`, `@expo-google-fonts/inter@0.4.2`
- `prettier-plugin-tailwindcss`

**Created config files:**

| File | Purpose |
|---|---|
| `tailwind.config.js` | All color, border-radius, font tokens in `theme.extend` |
| `src/app/global.css` | `@tailwind base/components/utilities` directives |
| `babel.config.js` | `nativewind/babel` preset + `jsxImportSource: "nativewind"` |
| `metro.config.js` | Wrapped with `withNativeWind()` |
| `nativewind-env.d.ts` | TypeScript types for NativeWind |

**`soldbay-app/src/app/_layout.tsx`**
- Loads fonts via `Font.loadAsync`:
  - Bricolage Grotesque 600/700/800 → `BricolageGrotesque-SemiBold`, `-Bold`, `-ExtraBold`
  - Inter 400/500/600 → `Inter-Regular`, `-Medium`, `-SemiBold`
- Shows splash screen until fonts loaded, then renders `<Stack />`
