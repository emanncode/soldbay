## API Route ‚Äî POST /api/waitlist

**File:** `soldbay-web/src/app/api/waitlist/route.ts`

- Accepts JSON: `{ role, name, email, university, level?, sellsWhat?, frequency?, categories[], pollAnswers[] }`
- Returns 400 if name, email, university, or role is missing/invalid
- Returns 409 if email already exists (catches Prisma P2002)
- Returns 201 `{ id }` on success
- Returns 500 on unexpected errors
